# Linux Terminal Basics
## Here we will basics Linux terminal commands (using [Ubuntu](http://ubuntu.com))

___
# ls

Syntax:
`ls dir`

(Listing `dir`)

Example:

`ls home`:

`user1 user2`


___

# cd

Syntax:
`cd dir`

(moving)

Example:

`user@device:~$ cd Desktop`

`user@device~/Desktop$ _`
